# Définition : 

- **Energie primaire :** énergie n'ayant subi aucune transformation après extraction 
    (houille, lignite, pétrole brut, gaz naturel, électricité d'origine hydraulique, 
    nucléaire, éolienne, géothermique, etc.). 
    
    
# Technologies : 

- Python3.5
- D3JS
